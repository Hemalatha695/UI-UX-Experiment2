# UI-UX-Experiment2

## Aim:
To design a basic mobile and web layout for an existing case study using Gestalt principles, UI elements, interaction behaviors, and usability principles. The design also includes a brand logo created in Canva
 following branding guidelines.
## Algorithm:
1.Select an existing case study and identify user requirements.
2.Create wireframes for both mobile and web layouts.
3.Apply Gestalt principles like proximity, similarity, and continuity in the design.
4.Add UI elements such as buttons, navigation bars, cards, and forms.
5.Develop interaction behaviors like hover effects, smooth navigation, and form validation.
6.Design a simple brand logo in Canva using suitable colors and typography.
7.Finalize and test the layouts for usability and responsiveness.
